Pointer Collection 1.0

Here's a couple of alternative pointer sets for OS/2. 

Included in this set:
Cherub Pointer Set
Sun Pointer Set

Installation:
1. Unzip the pointer sets with their folders and place these folder in the OS2\POINTERS directory. 
2. Load the pointer set from the Pointers setting found in the Mouse settings in System Setup.
3. Enjoy.

Note:
The hot point for the Cherub arrow pointer is the cherub's left hand, the hot point for the Sun arrow pointer is at the end of the sun's longer NW solar flare.
